# AliVolt Motors

AliVolt Motors is a **Tesla-inspired electric vehicle website** built with **React and Vite**.
The application allows users to explore electric cars, view their specifications, and navigate between detailed vehicle pages.

## Features

* Explore different electric vehicle models
* Dynamic car detail pages
* Responsive design for mobile and desktop
* Global state management using Redux Toolkit

## Technologies Used

* React
* Vite
* Redux Toolkit
* React Router
* Styled Components
