# AliVolt Motors ⚡

AliVolt Motors is a **Tesla-inspired electric vehicle website** built with **React and Vite**.
The application allows users to explore different electric cars, view their specifications, and navigate between detailed vehicle pages.

---

## 🚀 Features

* Tesla-style full screen car sections
* Explore different electric vehicle models
* Dynamic car detail pages
* Responsive design for mobile and desktop
* Smooth scroll animations
* Global state management using Redux Toolkit

---

## 🛠 Technologies Used

* React
* Vite
* Redux Toolkit
* React Router
* Styled Components

## ⚙️ Installation

Clone the repository:

git clone https://github.com/yourusername/alivolt-motors.git

Navigate to the project folder:

cd alivolt-motors

Install dependencies:

npm install

Start the development server:

npm run dev
